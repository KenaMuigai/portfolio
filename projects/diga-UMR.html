DiGA: Digital Health Decision Aid

collaboration project • 2024 - 2025

DiGA is a smart web platform that recommends certified digital health applications (DiGAs) to patients in Germany. Built as part of a collaborative project, it helps users find personalized solutions based on symptoms or health goals.

> Features

  • Semantic search using Sentence Transformers + Qdrant
  • Interactive DiGA quiz for guided recommendations
  • Secure login and personalized user sessions
  • FAQ-driven DiGA detail pages
  • Real-time search and category filtering

> Tech Stack

  • Frontend: Tailwind CSS, Bootstrap 5, Vanilla JS
  • Backend: Django (Python)
  • Semantic AI: Sentence Transformers + Qdrant
  • Database: SQLite (development), PostgreSQL-ready
  • Deployment: Microsoft Azure App Services

> My Role

I contributed to both backend and frontend development, supported integration of semantic search, designed patient-facing UI elements, and co-authored the documentation and user manual.
