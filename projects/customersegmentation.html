Customer Segmentation Using Recency, Frequency and Monetary (RFM) Analysis and K-Means Clustering Algorithm to improve strategies for SMEs.

project by joy • 2022

A Django-based web application that empowers SMEs to better understand and segment their customers through machine learning. This tool supports feature selection, cluster analysis using K-Means, and PDF report generation for business insights. A similar version of this project also allows for calculation of customer lifetime value (CLV9:)

> Features

  • CSV upload with real-time data inspection
  • Attribute filtering and statistics
  • Interactive clustering using K-Means
  • PDF report export with cluster summaries

> Machine Learning

  • RFM scoring (Recency, Frequency, Monetary)
  • K-Means clustering with customizable K

> Tech Stack

  • Django, Python, SQLite
  • JavaScript, HTML/CSS
  • Matplotlib, Pandas
